========================================
🚀 立即開始部署 - 快速指南
========================================

你現在要做的事情（按順序）：

第一步：上傳到 GitHub（5 分鐘）
========================================
1. 開啟 https://github.com/
2. 建立新 Repository（名稱：personnel-api）
3. 在專案目錄執行：

   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的帳號/personnel-api.git
   git push -u origin main

詳細步驟請看：上傳到GitHub.txt


第二步：部署到 Railway（10 分鐘）
========================================
1. 開啟 https://railway.app/
2. 使用 GitHub 登入
3. New Project > Deploy from GitHub repo
4. 選擇 personnel-api
5. + New > Database > PostgreSQL
6. 設定環境變數
7. 初始化資料庫
8. Generate Domain

詳細步驟請看：部署步驟_Railway.md


第三步：測試（5 分鐘）
========================================
1. 測試 API：
   https://你的URL/api/health
   https://你的URL/api/LoadPersonnelData

2. 更新 index.html 的 API URL

3. 測試前端功能
